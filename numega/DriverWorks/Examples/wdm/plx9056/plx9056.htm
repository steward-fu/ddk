Sample Driver for PLX9056 Development Board

[This is preliminary documentation and subject to change.]

Summary

This sample driver demonstrates how to program a PLX906 Development Board

Theory of Operation

DriverWorks library provides KPlx9056 class to help with programming PLX9056-based PCI boards. The sample driver demonstrates how to use this class to program PLX9056 board to perform the following operations:

  • Read/Write Local Bus Address Space in Direct Slave Mode
  • Read/Write Local Bus Address Space using DMA engine
  • Read/Write Mailbox Registers
  • Read/Write EEPROM

Building the Driver

First, build the DriverWorks libraries. Next build the plx9056 and plxcon projects.

Installing the Driver

First install the PLX9056 Development Board into a machine. Next time you boot Windows XP will detect the board and will ask you for an inf file. Point to plxdrv.inf file. You can use plxcon.exe utility to perform different operations with PLX9056 Board.
